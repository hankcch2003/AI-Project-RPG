# 運用生成式 AI 技術進行 RPG 遊戲開發專題

這是我在大學期間擔任組長主導的專題專案。我們致力於探索如何將**生成式 AI**整合進傳統的遊戲開發流程中，將團隊成員從單純的玩家提升為具備 AI 協作能力的開發者。

## 🚀 專案亮點
- **AI 協作開發**：利用 ChatGPT 進行劇本編寫與 Prompt 優化，並運用 DreamStudio 生成高品質的遊戲美術資源。
- **提示工程實踐**：深入研究 Prompt Engineering，確保 AI 生成的視覺風格（如 Anime 風格）在整個專案中保持高度統一。
- **團隊領導經驗**：擔任專題組長，負責任務分配、進度管理與團隊技術整合。

## 🛠 使用工具
- **開發引擎**：RPG Maker MV
- **生成式 AI**：ChatGPT (提示詞優化)、DreamStudio (圖像生成)
- **技術核心**：Prompt Engineering、影像處理、故事線邏輯設計

## 📝 製作心得
在這一年的開發過程中，我不僅掌握了 AI 工具的實務應用，更學會了如何擔任一個負責任的組長。從最初的技術驗證到最後的成果整合，我體會到「技術整合能力」與「團隊溝通」是完成優質專案的關鍵。
